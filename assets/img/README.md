# Dossier Images

Ce dossier contient les images décoratives de l'application.

## Images recommandées

### 1. Logo (logo.png)
- **Format** : PNG avec transparence
- **Dimensions recommandées** : 200-400px de largeur
- **Style** : Minimaliste, élégant
- **Couleurs** : Tons neutres (gris, noir) ou monochrome

### 2. Image de fond (background.jpg) - Optionnel
- **Format** : JPG ou PNG
- **Dimensions** : 1920x1080 ou supérieur
- **Style** : Image apaisante, nature, wellness
- **Opacité** : L'image peut être subtile et discrète

## Style recommandé

Pour rester fidèle à l'esthétique Buchinger Wilhelmi :
- Privilégiez des images épurées et minimalistes
- Tons clairs : blanc, gris, beige
- Évitez les couleurs vives ou contrastées
- Nature, zen, méditation, santé

## Exemples d'images appropriées

- Paysages naturels apaisants (montagnes, lacs)
- Éléments de nature (feuilles, eau)
- Symboles de santé et bien-être
- Formes géométriques minimalistes
- Textures douces (marbre, lin)

## Note

L'application fonctionnera parfaitement même sans ces images. Elles sont purement décoratives et améliorent l'expérience visuelle.

Si vous n'ajoutez pas de logo, la section du logo sera automatiquement masquée.
